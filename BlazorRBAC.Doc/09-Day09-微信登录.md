# Day 09 - 微信登录集成

> ⏱️ 预计时间：2小时

## 🎯 今日目标

- [ ] 了解 OAuth 2.0 流程
- [ ] 配置微信开放平台
- [ ] 实现微信登录回调
- [ ] 用户绑定逻辑

---

## 💻 核心流程

1. 用户点击"微信登录"
2. 跳转到微信授权页面
3. 微信回调返回 code
4. 后端用 code 换取 access_token
5. 获取用户信息并创建/绑定账号
